宝可梦◇熵增定律◇与死亡誓言

Posted on 2021-12-13 Edited on 2023-02-27 In Pokemon Views:

强大的宝可梦，

弱小的宝可梦，

那只是人的一己之见.

真正强大的训练家应该为了用自己喜欢的宝可梦获胜而努力才对哦.

你知道什么才是最重要的呢.

像以往那样前进吧，

冠军在等你哦. -------- 梨花

Preface

我知道你，你就是那个把宝可梦往死里使唤的训练师！

你算什么！没有宝可梦，你算什么！

你……到底伤害了多少精灵才走到今天这一步的？

我有话要对你说.

那是我在唐草镇第一次遇见你的时候.

从你的宝可梦那里听到的声音让我很震惊………

因为它说

喜欢你，

想和你在一起.

当时我不理解.

世界上居然有喜欢人类的宝可梦.

在那之前我还不知道有这样的宝可梦………

从那之后，随著我继续旅行，我的内心逐渐动摇………

因为我遇到的都是心灵相通、互相帮助的宝可梦和人类.

所以，我才想和你战斗，看看自己相信的东西究竟是什么………

我曾希望能同样作为英雄，与你面对面………

只考虑宝可梦………

不，连宝可梦也无法理解的我………

根本不可能战胜与众多宝可梦邂逅，身边还围绕著同伴的你………

冠军原谅我了………

但是今后我该怎么做，还是要我自己决定………

你说过，你有梦想.

那就去实现那个梦想吧！

实现美好的梦想，将它作为你所追求的真实！

那么………

再见了！ -------- N

RNG Abuse

  • 第三世代(RSE)
    • 宝可梦绿宝石及没有电的红蓝宝石实机乱数（不包含 ID 或 Egg）
    • 宝可梦有电的红蓝宝石实机乱数（不包含 ID 或 Egg）
    • 宝可梦火红叶绿实机乱数（不包含 ID 或 Egg）
  • 第四世代(DPPt HgSs)
    • 宝可梦四代实机乱数
  • 第五世代(BW BW2)
    • 贴吧有详细教程，需要注意，bw的seed0会极小程度波动（实操时可以看作不波动）而bw2的seed0可能会有好几个可能的取值，乱数时需要多加尝试，建议花时间通过统计来得到概率最大的seed0。 > 可能会有更加方便或者准确的方法，请读者自行去外网查阅资料。
  • 第六世代(XY OrAs)
    • 宝可梦六代实机乱数（破解机插件辅助 & 不包含蛋）
  • 第七世代(SM UsUm)
    • 贴吧有详细教程，丘丘尚未深入研究。
